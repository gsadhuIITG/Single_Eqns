The file `fast_phenotype_regime.c` implements numerical solutions for Equations 2 and 3, as well as the single-model Equation 12, within the fast-phenotype switching regime.


The MATLAB script 'tumor_volume_spatial_dist.m' provides temporal profiles of tumor volume for both the single-equation and two-equation models, along with the corresponding spatial distributions
of cell densities at time points t = 10, 25, 35, and 50.
